# Calculator (Python)

## Description

A Python calculator application that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Simple and easy-to-use interface

## Technologies Used

* Python

## How to Run

1. Clone this repository.
2. Open the project in your preferred Python IDE.
3. Run the main Python file.
4. Follow the on-screen instructions.

## Future Improvements

* Add a graphical user interface (GUI)
* Support advanced mathematical operations
* Add calculation history
